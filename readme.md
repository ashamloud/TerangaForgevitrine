# TerangaForge - L'IA Forgée au Sénégal

## 📋 Description

TerangaForge est l'agence digitale IA du réseau **BOKK**. Nous accompagnons les particuliers, chercheurs d'emploi, entrepreneurs et PME au Sénégal à travers des solutions numériques d'exception, alliant puissance technologique et identité locale.

## 🚀 Fonctionnalités Premium

### Pages de la Plateforme
- **Portail d'Accueil** : Design glassmorphic avec effets Aurora.
- **Nos Services** : Catalogue interactif (CV IA, Identité Visuelle, Web).
- **L'Histoire** : Notre vision et nos valeurs (Mission Teranga).
- **Contact & Support** : Intégration WhatsApp directe.
- **Cadre Légal** : CGU, Mentions et Confidentialité synchronisés.

### Services de Forge
- ✅ **Générateur de CV IA** : Mise en page professionnelle en 30s.
- ✅ **Design d'Excellence** : Logos, chartes graphiques et flyers.
- ✅ **Sites Vitrines Lucides** : Présence web moderne et rapide.
- 🚧 **Écosystème BOKK** : Intégration avec TerangaLearn et TerangaJob.

## 🛠️ Stack Technologique

- **Architecture** : HTML5 sémantique / Tailwind CSS JIT.
- **Design System** : Glassmorphism & Aurora Gradients.
- **Iconographie** : Lucide Icons (Dynamique).
- **Animations** : AOS (Animate On Scroll) & Custom CSS Keyframes.
- **Interaction** : JavaScript (Loud.js / Main.js for UI logic).
